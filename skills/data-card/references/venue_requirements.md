# What each venue requires

Confirm against the current call for papers — venues update these
year-to-year.

| Venue | Reproducibility | Datasheet | Model card | Artifact eval | Code release |
|---|---|---|---|---|---|
| NeurIPS | Checklist (mandatory) | D&B track only | Encouraged | No formal track | Strongly encouraged |
| ICML | Checklist (mandatory) | Optional | Optional | No | Strongly encouraged |
| ICLR | Repro statement (§) | Optional | Optional | No | Strongly encouraged |
| ACL/EMNLP/NAACL | Responsible NLP checklist + limitations § | Optional | Encouraged | No | Strongly encouraged |
| AAAI | Checklist | Optional | Optional | No | Encouraged |
| IJCAI | None formal | Optional | Optional | No | Encouraged |
| CVPR/ICCV/ECCV | None formal | Optional | Optional | No | Encouraged |
| ACM MM | None formal | Optional | Optional | No | Encouraged |
| SIGMOD | None main-track | Optional | Optional | **AE track** (separate) | Required for AE badge |
| VLDB | None main-track | Optional | Optional | **AE track** (separate) | Required for AE badge |
| ICDE | None main-track | Optional | Optional | AE some years | Encouraged |
| KDD | Checklist | Optional | Optional | AE track (some years) | Strongly encouraged |
| SIGIR | None formal | Optional | Optional | Repro track | Strongly encouraged |
| WWW | None formal | Optional | Optional | No | Encouraged |

## Special: NeurIPS Datasets & Benchmarks Track

Requires full datasheet (25 questions per Gebru et al.) plus a
dedicated paper structure. Different from main track.

## Special: SIGMOD/VLDB AE badges

Badges available: "Artifacts Available", "Artifacts Evaluated —
Functional", "Artifacts Evaluated — Reusable", "Results Reproduced",
"Results Replicated". Higher badges require more rigorous submission
packages.

## Special: ACL Limitations Section

ACL ARR now requires an explicit Limitations section (separate from
Discussion). Failure to include can cause desk reject. Limitations
must be honest, specific, and load-bearing.
